# Hi10Anime DL

A user-friendly GUI application for searching and downloading anime from Hi10Anime. Built with PyQt6, this tool provides an intuitive interface to browse anime titles and retrieve download links efficiently.

## Features

- **Search Anime**: Easily search for your favorite anime titles directly from the Hi10Anime website.
- **Download Links**: Retrieve direct download links for anime episodes with a single click.
- **Proxy Support**: Option to use system or environment proxies for accessing content.
- **Theme Support**: Switch between light and dark themes based on your preference or system settings.
- **Cross-Platform**: Primarily designed for Windows, with potential support for other platforms.

## Installation

### Prerequisites

- Python 3.8 or higher
- PyQt6
- Requests
- BeautifulSoup4
- Darkdetect

### Setup

1. Clone the repository:
   ```
   git clone https://github.com/Kurdeus/hi10anime-dl.git
   cd hi10anime-dl
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the application:
   ```
   python setup.py
   ```

### Download Executable (Windows)

You can download the standalone executable file for Windows directly from the [Releases](https://github.com/Kurdeus/hi10anime-dl/releases) page.
